# 功能介紹
這隻機器人可以：
* 偵測特定文字並傳送回應
* 使用斜線指令（slash command）
* 傳送回復方塊
* ~~測歐非驗血統~~ 抽卡模擬

這隻機器人不會：
* 放棄你
* 讓你失望
* 拋棄你
* 讓你哭泣
* 說謊傷害你
